# Atividade de Acessibilidade - Modo Claro e Modo Escuro
Descrição

Este projeto consiste em uma página web simples que demonstra a implementação de um recurso de acessibilidade: alternância entre modo escuro e modo claro. O objetivo é proporcionar uma melhor experiência visual para usuários com diferentes necessidades ou preferências.

Funcionalidades

Exibição de um título e um parágrafo de texto.

Dois botões para alternar entre modo escuro e modo claro.

Alteração da cor de fundo e do texto da página conforme o modo selecionado.

Tecnologias Utilizadas

HTML

JavaScript

Como usar

Abra o arquivo index.html em um navegador web.

Utilize os botões Modo Escuro e Modo Claro para alterar o tema da página.

Exemplo visual

Modo Escuro: fundo preto com texto branco.

Modo Claro: fundo branco com texto preto.

Melhorias futuras

Salvar a preferência do usuário no localStorage para manter o tema escolhido em visitas futuras.

Implementar transições suaves entre os modos.

Tornar o botão acessível via teclado e leitores de tela.
